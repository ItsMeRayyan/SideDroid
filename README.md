# SideDroid 📱

**A modern, open-source Android device manager powered by ADB and Fastboot. Built with Flutter and Rust for Windows, macOS, and Linux.**

![SideDroid Banner](assets/banner.png)

## ✨ About

SideDroid is a modern graphical interface for managing Android devices without needing to rely on the command line.

Built on top of Android Platform Tools, SideDroid makes ADB and Fastboot easier to use while still providing powerful tools for developers, enthusiasts, and Android power users.

## 🚀 Features

### 📱 ADB Tools

* Install APK files
* Uninstall applications
* View connected devices
* Device information
* Logcat viewer
* Screenshots
* Screen recording
* File management
* Wireless ADB support

### ⚡ Fastboot Tools

* Detect Fastboot devices
* Reboot controls
* View bootloader information
* Boot images
* Flash tools with safety checks

### 🛠 Platform Tools

* Automatic ADB and Fastboot support
* Platform Tools management
* Cross-platform compatibility

## 🎯 Why SideDroid?

Android tools are powerful, but they can be intimidating. SideDroid brings common ADB and Fastboot tasks into one clean interface while keeping advanced features available when you need them.

Whether you're installing apps, debugging a device, managing files, or working with Fastboot, SideDroid gives you the tools in one place.
